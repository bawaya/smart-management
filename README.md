# ניהול חכם — Smart Management

## מה זה?

מערכת ERP לניהול חברות שירותי שטח — ציוד, עובדים, חשבוניות, תקציב וכספים.

## טכנולוגיות

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Cloudflare D1 (SQLite)
- PWA

## התקנה

```bash
npm install
npx tsx src/lib/db/migrate.ts
npm run dev
```
